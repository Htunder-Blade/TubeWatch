# 实施计划

以下是分阶段计划；Phase 0 至 Phase 2 已实现，后续阶段仍只是计划。

- **Phase 0：项目骨架和频道视频读取。** 建立 Python 包，通过 Notebook 手动读取公开频道最近视频。
- **Phase 1：保存检测状态并判断新增视频。** 使用 SQLite 记录每个来源已发现的视频；首次全部视为新增，重复检查去重。（已实现）
- **Phase 2：新增视频调用 TubeScribe。** 通过固定版本的外部 Python 包显式处理 pending 视频，记录成功或失败，不自动重试。（已实现）
- **Phase 3：支持播放列表。** 将播放列表作为另一种视频来源。
- **Phase 4：处理失败重试。** 为 `failed` 设计有边界的重试策略；`no_subtitles` 默认保持终态，不自动重试。
- **Phase 5：定时运行与通知。** 在核心流程稳定后增加 Windows 定时执行和通知。
